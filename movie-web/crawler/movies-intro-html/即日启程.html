Set Off

　　旅居塞浦路斯的窝囊男人老崔本想发笔洋财后，回国光宗耀祖。谁料愿望还未达成，就被逼着回国离婚。匆忙归国的老崔赫然发现妻子不但是大着肚子，而且怀的竟还是自己发小朋友老魏的骨肉。面对前妻的冷嘲热讽，老崔惟有默默吞咽着心中的沮丧。他只想赶快回到塞浦路斯，因为他引以为后半生全部希望的、与人合开的中餐馆就要被合伙人卖掉。

　　天有不测风云，倒霉的老崔鬼使神差地意外遇到了一位“灾星”似的女孩小夏，从此开始了他噩梦般的日子……

　　老崔好心送酒醉的小夏回家，却被小夏误当作坏人赶出家门。为了讨回丢失行李和护照，老崔中了咣咣的圈套。原来小夏的男朋友大宝黑了同哥们咣咣、老五从债主装修经理唐金山处抢来的一笔钱美金，躲了起来。泼辣鬼机灵的小夏看准了老崔窝囊老实的性格，故意藏起老崔的护照，并以此为条件要老崔帮助自己拿回赃款送到躲藏起来的大宝手里。老崔和小夏历经周折拿到了赃款，发现钱是假的。一段时间的相处，小夏被忠厚的老崔感动，她主动把护照还给老崔，让他尽快离开这场是非……

　　谁知，唐金山为了追回藏着惊天秘密的假币，绑架了夏，要挟老崔。为救小夏准备登机回塞浦路斯的老崔毅然返回。紧要关头，老崔显示出从未有过的勇气，老崔舍命一搏救下小夏。自己却被装修经理唐金山等人逼到墙角……就在危机时刻，警方及时赶到救下老崔和小夏，并警方顺藤摸瓜，一举破获了唐金山预谋两年之久的盗窃某博物馆国宝案。

　　所有的麻烦终于解决了，老崔却不想再回到塞浦路斯，他能和小夏这对不打不相识的欢喜冤家未来的命运如何，相信大家的心里都有一个大团圆的结局。
